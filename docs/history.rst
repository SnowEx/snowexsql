* Project Created.

Hack Week 2021 (2021-07-15)
---------------------------

0.2.0 Repo Split (2022-06-20)

-----------------------------

* New columns were added to the LayerData table for flags
* Converted surveyors to observers
* Changed utm zone to be an integer
